# John Maus Project

This repository contains the final project for the *Information Science and Cultural Heritage* course (a.y. 2025/2026) of the Master's Degree in Digital Humanities and Digital Knowledge at Alma Mater Studiorum – University of Bologna.

## The Project

The project focuses on **John Maus**, an American musician, composer and academic. The aim is to represent information about John Maus through XML, TEI, RDF and Linked Data methodologies, documenting the entire workflow through a website and related files.

## Repository Contents

This repository contains:

* XML source file (`john_maus_XML.xml`)
* TEI encoding (`john_maus_TEI.xml`)
* XSLT transformation (`john_maus_TEI_to_HTML.xsl`)
* HTML output (`john_maus_TEI_HTML_output.html`)
* TEI to RDF transformation (`john_maus_TEI_to_RDF.py`)
* RDF/Turtle output (`john_maus_RDF_output.ttl`)
* Conceptual and theoretical models
* Final project website
